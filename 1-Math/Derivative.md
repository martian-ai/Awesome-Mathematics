# Common derivative
### Constant
+ C'=0

### Power
+ (x^n)'= nx^(n-1) 

### Trigonometric function
+ (sinx)' = cosx
+ (cosx)' = - sinx
+ (tanx)'=1/(cosx)^2=(secx)^2=1+(tanx)^2
+ (cotx)'=-1/(sinx)^2=-(cscx)^2=-1-(cotx)^2
+ (secx)'=tanx·secx
+ (cscx)'=-cotx·cscx
+ (arcsinx)'=1/(1-x^2)^1/2
+ (arccosx)'=-1/(1-x^2)^1/2
+ (arctanx)'=1/(1+x^2)
+ (arccotx)'=-1/(1+x^2)
+ (arcsecx)'=1/(|x|(x^2-1)^1/2)
+ (arccscx)'=-1/(|x|(x^2-1)^1/2)
### Hyperbolic function
+ (sinhx)'=coshx
+ (coshx)'=sinhx
+ (tanhx)'=1/(coshx)^2=(sechx)^2
+ (coth)'=-1/(sinhx)^2=-(cschx)^2
+ (sechx)'=-tanhx·sechx
+ (cschx)'=-cothx·cschx
+ (arsinhx)'=1/(x^2+1)^1/2
+ (arcoshx)'=1/(x^2-1)^1/2
+ (artanhx)'=1/(x^2-1) (|x|<1)
+ (arcothx)'=1/(x^2-1) (|x|>1)
+ (arsechx)'=1/(x(1-x^2)^1/2)
+ (arcschx)'=1/(x(1+x^2)^1/2)

### Exponential function
+ (e^x)' = e^x
+ (a^x)' = （a^x）lna （ln为自然对数）

### Logarithmic function
+ (Inx)' = 1/x（ln为自然对数）
+ (logax)' =x^(-1) /lna(a>0且a不等于1)
+ (x^1/2)'=[2(x^1/2)]^(-1)
+ (1/x)'=-x^(-2)

## The derivative of the product 
[f(x)g(x)]' = f(x)'g(x)+f(x)g(x)'
